# Multiple Object Detection using COCO-SSD in TensorFlow.js

This project is a browser-based Web AI application developed using TensorFlow.js and the COCO-SSD pre-trained model.


## Live Demo
https://ayannayak23.github.io/Object_Detection_COCO-SSD/

## Overview
- Client-side JavaScript handles all detection logic
- Uses COCO-SSD for real-time object detection
- Displays bounding boxes, labels, and confidence scores
- Includes performance and UI controls for better usability

## Deployment
This project is deployed using **GitHub Pages** and can be accessed via the live demo link above.
