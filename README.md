# Multiple Object Detection using COCO-SSD in TensorFlow.js

This project is a browser-based Web AI application developed using TensorFlow.js and the COCO-SSD pre-trained model.

## Requirements

Before running the project, make sure you have the following installed:

- Node.js (v16 or later recommended)
- npm (comes with Node.js)


## Setup Instructions

1. Download or clone this repository.
2. Open a terminal in the project root directory.
3. Navigate to the app folder:

```bash
   cd app
```

4. Install dependencies:

```bash
   npm install
```

## Running the Application

To start the web application, run the following command from inside the app directory:

```bash
   node app.js
```

Once the server is running, open your browser and go to: 

   `http://localhost:3000`


## Build Instructions

This project does not require a separate build step.  
All functionality runs directly using Node.js.
